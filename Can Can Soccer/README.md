# README Can Can Soccer

### Repository Content:
An editable copy of DPRG's Can Can Soccer competition rules is available in this directory. DPRG uses this file when making changes to the competition rules. It should **_not_** be considered the "official" rules. The "official" rules are located on DPRG's website at https://www.dprg.org/contest-rules/. Rule clarifications are located at https://www.dprg.org/contest-rule-clarifications/.
